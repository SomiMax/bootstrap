Esercizio Bootstrap

1. A cosa serve Bootstrap?

Risposta: Bootstrap è un framework CSS per creare siti web responsive e mobile-first con componenti predefiniti.

2. Cosa devi importare per creare una pagina web con Bootstrap?

Risposta:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

3. Cos'è un CDN? Che vantaggi comporta?

Risposta: CDN è una rete di server distribuiti.

Vantaggi:

  • Migliori performance
  • Riduzione carico sul server
  • Cache più efficiente

4. A cosa serve content="width=device-width, initial-scale=1"?

Risposta: Rende la pagina responsive:

  • width=device-width: viewport uguale al dispositivo
  • initial-scale=1: zoom al 100%

5. Che differenza c'è tra Container e Container Fluid?

Risposta:

  • Container: Larghezza fissa che cambia ai breakpoint
  • Container Fluid: Occupa sempre il 100% della larghezza

Container Fluid - 100% larghezza

Container Normale - larghezza fissa
